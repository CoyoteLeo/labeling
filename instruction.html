Relation Label
Match: The event from both recalled paragraph and retold paragraph describe the same thing, and there are no conflicts between each other. (i.e. they can use different words, but express the same meaning.), there are some examples below

PS:
he, asked, me
  • he => subject
  • asked => predciate
  • me => object
Conflict: In contrast to "Match", Conflict means the events describe the same thing, but the meaning is inconsistent. there are some examples below
recalled event retold event relation reason
No one, plays, sports you, do, sports Conflict Same event, but their meaning are conflict.
I, walk to, school I, drive to, school Conflict Same event, but their meaning are conflict.
(No relation): If there are not any relations between events, you can just leave it empty and continue to label the next pair.
In other words, you don't have to link the dots between the recalled and retold events.
